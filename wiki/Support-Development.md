# Support Development

Smart! is a passion project built for the Satisfactory community. Development, testing, documentation, support, and release work all take time.

Support is completely optional. If Smart! saves you time and you want to help keep development sustainable, Ko-fi contributions are appreciated:

- [Support Finalomega on Ko-fi](https://ko-fi.com/finalomega)

You can also support Smart! without donating:

- Report clear bugs on [GitHub Issues](https://github.com/majormer/SmartFoundations/issues).
- Join the [Smart! Discord](https://discord.gg/SgXY4CwXYw) for help, testing discussion, and feedback.
- Share screenshots, videos, and examples of what you build with Smart!.
- Help other players understand the current controls and features.
- Contribute fixes or documentation through pull requests.

No pressure. Every kind of support helps.
